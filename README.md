# Connect 4 AI Game

An intelligent Connect 4 game implemented in Python using adversarial search and machine learning techniques.

## AI Techniques Used
- Minimax with Alpha–Beta Pruning
- Heuristic Evaluation Functions
- Naive Bayes Classification
- Neural Network (MLP)
- Ensemble Scoring Framework

## Features
- Human vs AI gameplay
- Dynamic board sizes (6×6, 6×7)
- Emoji-based board visualization
- Strong defensive and offensive AI behavior

## Technologies
- Python
- Scikit-learn
- NumPy

## How to Run
```bash
python final_connect4.py
